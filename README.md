# Notes App

A simple Notes App built using HTML, CSS, and JavaScript.

## Features

- Add new notes
- Delete notes
- Notes are saved in browser storage
- Notes remain available after page refresh
- Simple and user-friendly interface

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage

## How It Works

1. Type a note in the text area.
2. Click the **Add Note** button.
3. The note will be displayed below.
4. Notes are automatically saved in the browser.
5. Click **Delete** to remove a note.

## Concepts Used

- DOM Manipulation
- Functions
- Arrays
- Loops
- Event Handling
- Local Storage
- JSON.stringify()
- JSON.parse()

## Project Purpose

This project was created to practice JavaScript fundamentals and browser storage concepts while building a useful real-world application.
